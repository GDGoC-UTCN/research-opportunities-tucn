# TUCN Research Opportunities 🎓

A modern web platform connecting students with professors at the **Technical University of Cluj-Napoca (UTCN)** for research and academic projects.

See the [tucn-research-opportunities](tucn-research-opportunities/) folder for the frontend React application and complete documentation.

## Features:
- 🔵 **Student Flow:** Browse opportunities, view requirements, and submit applications.
- 🎓 **Professor Flow:** Create new research projects, view a personalized dashboard, and review applicants.
- 🎨 **Modern UTCN Design:** Powered by Tailwind CSS v4, Framer Motion, and React.

### Quick Start:
```bash
cd tucn-research-opportunities
npm install
npm run dev
```
Open `http://localhost:3000` to preview the mock application!
